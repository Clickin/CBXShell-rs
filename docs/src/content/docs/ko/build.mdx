## 빠른 요약

- **필수 도구:** Rust stable + Windows 10 SDK + MSVC 빌드 도구
- **빠른 빌드:** `cargo build --release`
- **타깃:** `x86_64-pc-windows-msvc`, `aarch64-pc-windows-msvc`
- **테스트:** `cargo test`

## 요구 사항

- Rust 1.70+ (stable)
- Windows 10 SDK 이상
- Visual Studio Build Tools (MSVC toolchain)

## 빌드 명령

```bash
git clone https://github.com/Clickin/CBXShell-rs.git
cd CBXShell-rs
cargo build --release
```

아키텍처별 빌드:

```bash
cargo build --release --target x86_64-pc-windows-msvc
cargo build --release --target aarch64-pc-windows-msvc
```

테스트 실행:

```bash
cargo test
```
