## 빠른 요약

- **로그 파일:** `%TEMP%\\cbxshell_debug.log`
- **추천 명령:** `Get-Content ... -Wait`로 실시간 확인
- **용도:** 썸네일 생성/아카이브 처리 문제 진단

CBXShell은 문제 진단을 위해 임시 파일에 디버그 로그를 기록합니다:

```cmd
C:\Users\<username>\AppData\Local\Temp\cbxshell_debug.log
```

PowerShell에서 실시간 확인:

```powershell
Get-Content "$env:TEMP\cbxshell_debug.log" -Wait
```
