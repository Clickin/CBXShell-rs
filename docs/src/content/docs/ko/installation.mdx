## 빠른 요약

- **무엇인가요:** 아카이브 썸네일을 제공하는 Windows Explorer 핸들러
- **지원 포맷:** `.cbz/.cbr/.cb7`, `ZIP/RAR/7z`, WebP/AVIF
- **요구 사항:** Windows 10/11, 관리자 권한
- **3단계 설치:** 다운로드 → 설치 → Explorer 재시작
- **삭제:** 제거 프로그램 또는 `cbxshell.dll` 등록 해제

## 옵션 1: NSIS 설치 프로그램 (GitHub Releases)

GitHub Releases 페이지에서 최신 설치 프로그램을 다운로드하세요. (기본 배포 채널)

- x64: `CBXShell-rs-Setup-x.x.x-x64.exe`
- ARM64: `CBXShell-rs-Setup-x.x.x-ARM64.exe`

PowerShell에서 시스템 아키텍처 확인:

```powershell
$env:PROCESSOR_ARCHITECTURE
```

설치 단계:
1. 관리자 권한으로 설치 프로그램 실행
2. 활성화할 파일 형식 선택 (CBZ, CBR, CB7, ZIP, RAR, 7Z)
3. 안내에 따라 Windows Explorer 재시작

## 옵션 2: 수동 설치 (고급)

```cmd
cargo build --release
cd target\release
regsvr32 cbxshell.dll
```

등록 해제:

```cmd
regsvr32 /u cbxshell.dll
```

Explorer 재시작:

```cmd
taskkill /f /im explorer.exe
start explorer.exe
```
