## 옵션 1: Microsoft Store (권장)

Microsoft Store에서 CBXShell을 설치하면 자동 업데이트를 받을 수 있습니다.

요구 사항:
- Windows 11 21H2 이상
- 약 5 MB 디스크 공간

## 옵션 2: NSIS 설치 프로그램 (GitHub Releases)

GitHub Releases 페이지에서 최신 설치 프로그램을 다운로드하세요.

- x64: `CBXShell-rs-Setup-x.x.x-x64.exe`
- ARM64: `CBXShell-rs-Setup-x.x.x-ARM64.exe`

PowerShell에서 시스템 아키텍처 확인:

```powershell
$env:PROCESSOR_ARCHITECTURE
```

설치 단계:
1. 관리자 권한으로 설치 프로그램 실행
2. 활성화할 파일 형식 선택 (CBZ, CBR, CB7, ZIP, RAR, 7Z)
3. 안내에 따라 Windows Explorer 재시작

## 옵션 3: 수동 설치 (고급)

```cmd
cargo build --release
cd target\release
regsvr32 cbxshell.dll
```

등록 해제:

```cmd
regsvr32 /u cbxshell.dll
```

Explorer 재시작:

```cmd
taskkill /f /im explorer.exe
start explorer.exe
```
