## MSIX 패키징 (Microsoft Store)

```powershell
.\build_msix.ps1 -Configuration Release -Platform x64
```

출력:
- `dist\msix\CBXShell_5.0.0.0_x64.msix`

요구 사항:
- Windows SDK 10.0.22621.0 이상
- Store 제출용 코드 서명 인증서

## NSIS 설치 프로그램 (GitHub Releases)

```powershell
.\build_nsis.ps1 -Configuration Release
```

출력:
- `dist\CBXShell-Setup-5.0.0.exe`

요구 사항:
- NSIS 3.x 이상
