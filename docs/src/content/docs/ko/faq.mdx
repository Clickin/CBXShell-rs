## 빠른 요약

- **썸네일이 안 보임:** Explorer 재시작, 썸네일 캐시 삭제, 등록 상태 확인
- **아카이브 미지원:** ZIP/RAR/7z 또는 `.cbz/.cbr/.cb7`인지 확인
- **WebP/AVIF 문제:** 최신 릴리스로 업데이트 후 로그 확인
- **로그 위치:** `%TEMP%\\cbxshell_debug.log`

## Explorer에서 썸네일이 보이지 않아요

**해결:** Explorer 재시작 후 썸네일 캐시를 정리하세요.

1. Explorer 종료 (`taskkill /f /im explorer.exe`)
2. Explorer 재시작 (`start explorer.exe`)
3. 필요 시 Windows 설정에서 썸네일 캐시 삭제

[설치](/ko/installation/)와 [로깅](/ko/logging/) 문서를 참고하세요.

## 일부 아카이브만 썸네일이 표시돼요

**해결:** 아카이브 내 이미지 포맷과 암호화 여부를 확인하세요.

- 지원 아카이브: `.cbz/.cbr/.cb7`, ZIP, RAR, 7z
- 지원 이미지: JPEG/PNG/GIF/BMP/TIFF/ICO + WebP + AVIF

특정 파일에서 오류가 나면 로그를 확인하세요.

## WebP/AVIF 썸네일이 느리거나 실패해요

**해결:** 최신 릴리스를 설치하고 이미지가 손상되지 않았는지 확인하세요.

WebP/AVIF 디코딩은 `image` crate를 사용하며 손상된 파일에서 실패할 수 있습니다.
`%TEMP%\\cbxshell_debug.log`에서 디코딩 오류를 확인하세요.

## 수동으로 등록 해제하려면?

**해결:** 관리자 권한 CMD에서 `regsvr32 /u cbxshell.dll` 실행.

[설치](/ko/installation/) 문서에 전체 절차가 있습니다.
