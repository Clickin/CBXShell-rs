## 완료됨

- IThumbnailProvider 썸네일 핸들러
- 멀티 포맷 아카이브 지원 (ZIP, RAR, 7z)
- 최신 이미지 포맷 (WebP, AVIF)

- 사용자 단위 설치 지원
