구성 관리자를 사용하면 레지스트리를 직접 수정하지 않고도
파일 형식별 핸들러 활성화와 정렬 동작을 설정할 수 있습니다.

## 실행

```cmd
CBXShell.exe --manager
```

## 기능

- 파일 형식별 핸들러 활성화/비활성화
- 아카이브 형식 선택 (CBZ, CBR, CB7, ZIP, RAR, 7Z)
- 알파벳 순 vs 발견 순 정렬 설정
