---
title: Configuration Manager
description: Manage handlers and sorting options.
---

The configuration manager lets you enable handlers per file type and configure
sorting behavior without manual registry edits.

## Launch

```cmd
CBXShell.exe --manager
```

## Features

- Enable/disable handlers per file type
- Choose archive formats (CBZ, CBR, CB7, ZIP, RAR, 7Z)
- Configure alphabetical vs. discovery order sorting
