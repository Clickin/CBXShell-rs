description: Install CBXShell from Store, installer, or manually.
---

## Option 1: Microsoft Store (Recommended)

Install CBXShell from the Microsoft Store for automatic updates.

Requirements:
- Windows 11 21H2 or later
- About 5 MB of disk space

## Option 2: NSIS Installer (GitHub Releases)

Download the latest installer from the GitHub releases page.

- x64: `CBXShell-rs-Setup-x.x.x-x64.exe`
- ARM64: `CBXShell-rs-Setup-x.x.x-ARM64.exe`

Check your system architecture in PowerShell:

```powershell
$env:PROCESSOR_ARCHITECTURE
```

Installation steps:
1. Run the installer as administrator.
2. Choose file formats to enable (CBZ, CBR, CB7, ZIP, RAR, 7Z).
3. Restart Windows Explorer when prompted.

## Option 3: Manual installation (Advanced)

```cmd
cargo build --release
cd target\release
regsvr32 cbxshell.dll
```

To unregister:

```cmd
regsvr32 /u cbxshell.dll
```

Restart Explorer to reload handlers:

```cmd
taskkill /f /im explorer.exe
start explorer.exe
```
