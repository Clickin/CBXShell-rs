## NSIS installer (GitHub Releases)

```powershell
.\build_nsis.ps1 -Configuration Release
```

Output:
- `dist\CBXShell-Setup-5.0.0.exe`

Requirements:
- NSIS 3.x or later

## Chocolatey package

Generate a Chocolatey package that bundles the NSIS installers:

```powershell
.\build_chocolatey.ps1 -BuildInstallers
```

Output:
- `dist\cbxshell-rs.<version>.nupkg`

Requirements:
- Chocolatey CLI (`choco`)
- NSIS 3.x or later
